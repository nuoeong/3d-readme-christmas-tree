## 🧐 Motivation

GitHub 프로필을 꾸미기 위해 많은 분들이 사용하는 `github-readme-stats`나 `readme-typing-svg` 같은 인기 서비스들이 트래픽 과부하로 인해 이미지가 깨지거나 서버가 다운되는 현상을 종종 겪습니다.

라는 생각으로, 외부 의존성 없이 **직접 Vercel에 호스팅하여 안정적으로 동작하는** 3D 크리스마스 트리를 만들게 되었습니다. <br />

<br/>

## ✨ Features

- **3D Animation**: CSS 3D Transform을 활용한 부드러운 회전 효과
- **Neon Glow**: 형광색 네온 사인 효과 (Red & Green)
- **Snow Effect**: 배경에 내리는 낭만적인 눈송이 애니메이션
- **Lightweight**: 무거운 외부 라이브러리 없이 순수 **SVG + CSS**로만 구현하여 매우 빠릅니다.
- **Stable**: 본인의 Vercel 계정에 직접 배포하므로 타 서비스의 서버 이슈에 영향받지 않습니다.

---

## 🎄 나만의 트리 만들기

**Vercel**을 통해 **단 1분 만에** 무료로 배포할 수 있습니다.
더 이상 외부 Vercel 주소를 쓰지 않아요!

### Step 1. Vercel로 복제하기
아래 **Deploy** 버튼을 클릭하면 이 프로젝트가 여러분의 Vercel 계정으로 자동 복제됩니다.

<div align="center">

👇👇

<a href="https://vercel.com/new/clone?repository-url=https://github.com/yooncandooit/github-readme-christmas-tree">
  <img src="https://vercel.com/button" alt="Deploy with Vercel"/>
</a>

</div>

### Step 2. 프로젝트 생성
1. Vercel에 GitHub 계정으로 로그인합니다.
2. `Create Git Repository` 화면이 나오면, **Repository Name**을 입력하고 (예: `my-christmas-tree`) **Create** 버튼을 누릅니다.
3. 잠시 기다리면 배포가 완료됩니다! 🎉

### Step 3. 도메인 주소 확인
1. 배포가 완료된 후 **"Go to Dashboard"** 버튼을 클릭하세요.
2. 화면 우측 상단의 **Domains** 부분에 있는 주소를 복사하세요.
   - 예시: `https://my-christmas-tree.vercel.app`

<br/>

## 🚀 Quick Start (적용하기)

위에서 복사한 도메인 주소 뒤에 `/api`만 붙여서, 아래 코드를 여러분의 `README.md`에 붙여넣으세요.

```markdown
<div align="center">
  <img src="https://(복사한-Vercel-주소)/api" alt="Christmas Tree" />
</div>
```
**Tip**: 만약 코드를 수정했는데 이미지가 바뀌지 않는다면, 주소 뒤에 ?v=1 처럼 숫자를 붙여주세요. GitHub 캐시 갱신용으로 붙이는 쿼리키이기 때문이에요! (예시: src=".../api?v=1")
